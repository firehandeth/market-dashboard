# 🐭 大鼠市场风控仪表盘 | Market Risk Dashboard

A comprehensive market risk dashboard based on **@investingluc's 5-Dimension Framework** for trading conviction × environment quality = position sizing.

## Quick Start

### View the Dashboard
```bash
# Option 1: Serve via HTTP
cd /home/ubuntu/.openclaw/workspace
python3 -m http.server 8765
# Then open: http://localhost:8765/dashboard/
```

Or simply open `dashboard/index.html` directly in your browser (works offline with embedded sample data).

### Update Market Data
```bash
cd /home/ubuntu/.openclaw/workspace
python3 scripts/market-dashboard.py
```

The script fetches 6 months of OHLCV data from Yahoo Finance and generates `memory/market-dashboard.json`.

## Architecture

### Files
- **`dashboard/index.html`** — Single-file, no-build dashboard (799 lines)
  - Embedded CSS (cyberpunk/terminal dark theme)
  - Pure vanilla JavaScript
  - Auto-refreshes every 5 minutes
  - Works with `file://` protocol (offline fallback with sample data)
  - Position sizing calculator with Conviction slider

- **`scripts/market-dashboard.py`** — Data generator
  - Fetches real-time prices from Yahoo Finance
  - Calculates 50/200 SMAs, RSI, momentum metrics
  - Scores all 5 dimensions
  - Outputs comprehensive JSON

- **`memory/market-dashboard.json`** — Data source
  - Auto-generated by the Python script
  - Used by dashboard for real-time rendering

## 5 Dimensions Explained

### 1️⃣ Current Environment (25% weight)
**Macro health check:** SPY/QQQ trends, risk appetite (BTC), yield curve, USD strength, safe-haven demand (Gold)
- Sub-metrics: 50SMA/200SMA trends, 1W/1M/3M changes
- Copper/Gold ratio as economic health indicator
- BTC momentum as risk-on/off signal

**Score:** % of bullish indicators above key SMAs

### 2️⃣ Relative Strength of Leaders (20% weight)
**Top 10 stocks health:** NVDA, META, AAPL, MSFT, AVGO, GOOGL, AMZN, TSLA, AMD, CRM
- Metrics: Price above 50SMA? RSI overbought/oversold? Momentum?
- Visual: Color-coded chips (green=bullish, red=bearish)

**Score:** % of leaders above 50SMA + % with positive 1M momentum

### 3️⃣ VIX Behavior (20% weight)
**Fear gauge & volatility structure**
- Zones: < 15 (complacent) / 15-20 (normal) / 20-30 (elevated) / >30 (fear)
- VIX vs 50SMA trend
- Term structure (VIX vs VIX3M): Contango (healthy) vs Backwardation (stress)
- Volatility clustering: Count of >2% SPY daily moves (last 10 days)

**Score:** VIX level, direction, term structure health

### 4️⃣ Breadth / Participation (20% weight)
**Sector health across 11 GICS groups**
- XLK (Tech), XLF (Financials), XLE (Energy), XLV (Healthcare), XLI (Industrials)
- XLP (Staples), XLU (Utilities), XLY (Discretionary), XLB (Materials), XLRE (Real Estate), XLC (Comm Services)
- Metrics: % above 50SMA, % positive 1M, best/worst performer

**Score:** Breadth participation %

### 5️⃣ Macro Backdrop (15% weight)
**Manual/qualitative assessment**
- Known macro factors (tariffs, Fed policy, geopolitics)
- Key upcoming events calendar
- **Editable notes** (stored in browser localStorage)
- **Override slider:** Adjust macro score 0-100% if you disagree with default

**Default:** 25% (conservative bias). Use the slider to override.

## Position Sizing Logic

**Overall Score + Thesis Conviction = Effective Position Size**

```
Conviction = 1-10 slider
Effective Conviction = Round(Conviction × Overall Environment Score %)
```

### Position Map
| C | Risk % | Description |
|---|--------|---|
| C1-4 | 0% | No trade — sit out |
| C5 | 0.9% | Minimum position |
| C6 | 1.5% | Light position |
| C7 | 3% | Standard (weekly cap) |
| C8 | 5% | Heavy |
| C9 | 10% | High conviction |
| C10 | 25% | Max conviction |

**Example:**
- Environment: 50% (neutral)
- Your thesis conviction: C7
- Effective: C7 × 50% = C3.5 → No trade recommendation
- But if environment = 70%, then C7 × 70% = C4.9 → Still hold back

## Customization

### Change Dimension Weights
Edit the `weights` object in Python script (~line 310):
```python
weights = {
    "1_environment": 0.25,
    "2_leaders": 0.20,
    "3_vix": 0.20,
    "4_breadth": 0.20,
    "5_macro": 0.15,
}
```

### Modify Position Sizing Table
Edit `SIZING_MAP` in the HTML dashboard:
```javascript
const SIZING_MAP = {
  5: {risk: '0.9%', pct: 0.9, desc: '最小仓位'},
  7: {risk: '3%', pct: 3, desc: '标准仓位 (周上限)'},
  // ...
};
```

### Add/Remove Leaders or Sectors
Edit the Python script:
```python
LEADER_SYMBOLS = ["NVDA", "META", ..., "YOUR_SYMBOL"]
SECTOR_SYMBOLS = ["XLK", "XLF", ..., "YOUR_ETFS"]
```

## Usage Tips

1. **Run script daily/hourly:**
   ```bash
   0 * * * * cd /home/ubuntu/.openclaw/workspace && python3 scripts/market-dashboard.py
   ```

2. **Use Macro Override when:**
   - Known Black Swan event (earnings surprise, geopolitical escalation)
   - Your thesis is independent of market environment (e.g., pure sector play)
   - Calendar shows high-impact event today

3. **Position Sizing Formula:**
   - Let environment be your **floor** (don't trade C7 in C3 environment)
   - Add your conviction **on top**
   - Accept that effective position shrinks in weak environments

4. **Track Performance:**
   - Log each trade with entry environment score
   - Review: Did I respect the environment framework?
   - Adjust conviction calibration over time

## Troubleshooting

**Dashboard shows "Loading..." forever?**
- Check: Is `memory/market-dashboard.json` readable?
- Run: `python3 scripts/market-dashboard.py` to regenerate
- Open console (F12) for errors

**API calls failing (timeouts)?**
- Yahoo Finance connection issue — check internet
- Switch to cached data: reload dashboard after 5 min

**Data stale?**
- Manual refresh: `python3 scripts/market-dashboard.py`
- Or wait for auto-refresh (5 min interval)

**On file:// protocol?**
- Dashboard auto-loads embedded sample data
- For live data, serve via HTTP: `python3 -m http.server 8765`
